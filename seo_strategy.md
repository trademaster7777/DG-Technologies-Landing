# SEO Strategy — D2G Technology

## Site type
Single-page React/Vite SPA marketing site for D2G Technology, a small-business website builder that works entirely by phone.

## In scope
- The public landing page served at `/` (`artifacts/dg-technologies`)
- All sections: Hero, Problem, HowItWorks, WhyUs, Pricing, Testimonials, Upsell, FAQSection, FinalCTA

## Out of scope
- API server (`artifacts/api-server`) — not public-facing HTML
- Admin / authenticated pages (none exist currently)

## Target audience
Small business owners, entrepreneurs, solopreneurs who want a professional website built quickly without technical knowledge.

## Primary keywords
- Affordable small business websites
- Small business website design
- Website builder for small businesses
- Professional websites for solopreneurs

## Rendering mode
React SPA (Vite). All page content is client-side rendered. Static `index.html` shell contains title, meta description, OG tags, canonical, and favicon — but no body copy, headings, or structured data visible to crawlers in the initial HTML response.
